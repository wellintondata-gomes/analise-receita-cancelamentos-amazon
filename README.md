# 📊 Análise de Receita e Cancelamentos | E-commerce Amazon

Projeto de análise de dados desenvolvido em Power BI com foco na identificação de prejuízos operacionais, análise de cancelamentos e geração de insights estratégicos para tomada de decisão.

---

# 🎯 Objetivos do Projeto

- Identificar categorias com maior índice de prejuízo
- Analisar taxa de cancelamentos/devoluções
- Avaliar comportamento de receita ao longo do tempo
- Encontrar produtos com maior geração de receita
- Propor estratégias de crescimento e redução de perdas

---

# 🛠 Ferramentas Utilizadas

- Power BI
- DAX
- Power Query
- Excel / CSV

---

# 📌 Principais KPIs

- Receita Bruta
- Receita Válida
- Receita Perdida
- Taxa de Cancelamento
- Ticket Médio

---

# 📈 Principais Insights

- Electronics concentra maior prejuízo operacional
- iPhone 14 lidera geração de receita válida
- Receita apresenta queda em 2025
- Taxa de cancelamento próxima de 40%

---

# 🚀 Estratégia Proposta

A estratégia foi estruturada em duas frentes:

1. Expandir categorias e produtos com maior geração de receita
2. Reduzir prejuízos operacionais nas categorias mais problemáticas

O objetivo consiste em maximizar retorno utilizando categorias já rentáveis enquanto os gargalos operacionais são tratados progressivamente.
